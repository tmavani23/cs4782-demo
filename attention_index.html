Attention Heads

A transformer can run many attention heads in parallel — each one asks a different question about the sentence. Think of it like having several people read the same sentence simultaneously, each looking for something different. Choose how many heads to use, then click one to view its pattern:

Real models: GPT-2 uses 12 · BERT uses 12–16 · GPT-4 likely uses 96+

Notice how each head produces a different attention pattern over the same sentence. In a real model, the outputs of all heads are combined — giving the model richer understanding than any single head alone.

Causal Mask GPT-style
GPT generates text left-to-right, so when predicting word i, it must not "peek" at words that come after it. The causal mask enforces this by blocking all attention to future words.
✅ Mask OFF (BERT-style)
Every word can see every other word. Good for understanding full sentences.
🚫 Mask ON (GPT-style)
Each word can only attend to earlier words. Future tokens are blocked. Good for generation.

Turn on the mask and notice the ✕ symbols appear in the top-right of the attention matrix — those connections are blocked.
